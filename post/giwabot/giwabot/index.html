    脚本作用：

    • 创建giwabot unit（先删除原先的，并停止原先正在运行的giwabot，再创建新的并运行）
    • 脚本会使能giwabot unit，保证下次开机自动运行

    giwabot@raspberrypi:~/Documents/solar-cleanbot-vision $ cat /usr/lib/systemd/system/giwabot.service
[Unit]
Description=Giwabot
#After=network-online.target

[Service]
Type=forking
User=giwabot
Group=giwabot
ExecStart=/home/giwabot/Documents/solar-cleanbot-vision/onbootstartup.sh
StartLimitInterval=0s
Restart=always
RestartSec=1

[Install]
WantedBy=multi-user.target

  2. 后续常用命令

    1. 开机自动运行后，关闭服务 sudo systemctl stop giwabot
    2. 设置开机不自动启动 sudo systemctl disable giwabot
    3. 手动开启服务 sudo systemctl start/restart giwabot
    4. 查看日志 journalctl -xu giwabot

代码篇

  1. 分支说明
    1. master
      1. 原先架构，协程方式运行。
      2. 手动运行命令： python -O hough.py
      3. 开机自动运行命令： sudo bash install-me.sh
    2. refactor
      1. 想对于master分支做的重构，多进程方式运行。
      2. 手动运行命令：python -o main.py
      3. 开机自动运行命令： sudo bash install-me.sh
